# Jason Lee — Personal Academic Website

A lightweight, responsive personal website built for GitHub Pages.

## Preview locally

Open `index.html` directly, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Publish with GitHub Pages

1. Create the public repository `JasonLee-chips.github.io`.
2. Push these files to the repository's `main` branch.
3. Visit `https://JasonLee-chips.github.io`.

GitHub Pages user sites are normally published automatically from the repository's root.

## Customize

Edit `index.html` to replace the introductory text, research interests, projects, and contact details. Add a `CNAME` file containing `jasonlee.chips` only after the custom domain's DNS records are configured.
